# AI Resume Analyzer

An AI-powered Resume Analyzer developed using Python and Streamlit. The application extracts text from uploaded resume PDFs, identifies technical skills, calculates a resume score, and recommends additional skills to improve employability.

## Features

* Upload Resume PDF
* Automatic Skill Extraction
* Resume Scoring System
* Skill Recommendations
* User-Friendly Interface

## Technologies Used

* Python
* Streamlit
* PDFPlumber
* Pandas

## Project Structure

AI-Resume-Analyzer

├── app.py

├── skills.py

## Installation

```bash
pip install -r requirements.txt
```

## Run the Application

```bash
streamlit run app.py
```
